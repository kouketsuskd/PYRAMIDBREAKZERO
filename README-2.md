# PYRAMID BREAK ZERO - DAYBREAK X

必殺技・演出・進行を再構築した版。

## 会話
- 通常攻撃、通常の敵攻撃、道具、防御、交代では会話を出さない
- 会話は「TALK」、ボス形態変化、横スクロール、ビーム、封印など新しいギミック発生時のみ

## 必殺技
- トシユキ：時計の針が12時に重なる瞬間を3回タップ
- カズアキ：左から右へ3回スワイプ
- マッピー：ピラミッドの頂点を1→2→3の順にタップ
- ケイタ：舞う火の粉を5つタップ
- イサジ：中央の「鬼」を長押しして力を制御
- リョウゴ：最初に光った未来を覚えて選択

## 演出
- 必殺技ごとの専用背景、紋章、発光、残像、成功演出を追加
- 必殺技の説明を画面上部へ固定し、操作対象と重ならないよう整理
- 成功後は専用のシネマティック画面を表示
