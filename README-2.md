# PYRAMID BREAK ZERO - DAYBREAK X.1

横スクロール・必殺技の完全修正版。

## 横スクロール
- ギミック終了会話をタップできなかった原因を修正
- 会話入力をすべてのギミック操作より優先
- 落下するとHPが12減り、チェックポイントへ即復帰
- 一度落下すると穴が「記憶の橋」に変わり、必ずゴールできる
- ゴール到達時は自動クリア。追加の操作は不要
- ジャンプ判定を緩和

## 必殺技
- 全キャラクターの必殺技にダメージを追加
- トシユキのザ・ワールドは時間停止ダメージ＋回復
- 時間停止成功後は敵ターンをスキップ
- 時計と停止波紋の専用演出を追加
- マッピーとケイタの回復型必殺技にも反射・炎ダメージを追加
- 必殺技ダメージを大きく画面表示
